# unified_game_services_game_center

Apple Game Center provider for
[`unified_game_services`](https://pub.dev/packages/unified_game_services).

> **Status: deferred (not yet implemented).** Game Center is the Obj-C GameKit
> framework; a pure-Dart path would go through `dart:ffi` +
> [`package:objective_c`](https://pub.dev/packages/objective_c) on Apple
> platforms. This package is a placeholder pending that work.

For the available providers today, see
[`unified_game_services`](https://pub.dev/packages/unified_game_services),
`unified_game_services_steam` and `unified_game_services_gamejolt`.
